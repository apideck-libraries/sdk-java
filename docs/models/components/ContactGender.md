# ContactGender

The gender of the contact.

## Example Usage

```java
import com.apideck.unify.models.components.ContactGender;

ContactGender value = ContactGender.MALE;

// Open enum: use .of() to create instances from custom string values
ContactGender custom = ContactGender.of("custom_value");
```


## Values

| Name     | Value    |
| -------- | -------- |
| `MALE`   | male     |
| `FEMALE` | female   |
| `UNISEX` | unisex   |
