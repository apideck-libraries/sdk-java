# PosPaymentWalletStatus

The status of the wallet payment. The status can be AUTHORIZED, CAPTURED, VOIDED, or FAILED.


## Values

| Name         | Value        |
| ------------ | ------------ |
| `AUTHORIZED` | authorized   |
| `CAPTURED`   | captured     |
| `VOIDED`     | voided       |
| `FAILED`     | failed       |
| `OTHER`      | other        |
